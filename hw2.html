CNIT 133 - Homework 2

Part 1

Part 1 demonstrates the ability to use document.write in Javascript.


Part 2a

Part 2a demonstrates the ability to use a form to collect three numbers from the user and calculate the sum, average, product, maximum, and minimum. If the user does not enter valid numbers this page displays an error message in the textarea box. There is also a button to fade the results area.


Part 2b

Part 2b demonstrates the ability to use a form to collect three numbers from the user and calculate the sum, average, product, maximum, and minimum. If the user does not enter valid numbers this page displays an alert. There is also a button to fade the results area.


Part 3

Part 3 demonstrates the ability to use a form to collect five numbers from the user and determine how many positive, negative, and zero values were entered. I utilized the array and for loop to increment through the 5 numbers and count the number of positive, negative, and zero values.


Extra Credit

The Extra Credit displays a table and form using document.write, converts USA dollars to other currencies, and inlcudes formatting elements such as a sliding panel for instructions and a style for the autofocus in a form.
